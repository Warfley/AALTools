Quelltexteditor ››

Der Texteditor ist ein Editor zum bearbeiten von .aal1 und .apr Dateien.


Der Editor verfügt über:

Farbige Syntaxhervorhebung

Suchen/Ersetzen

Hilfe zu Funktionen

CodeJumps zu Definitionen

Code Formatter zum Formatieren

Automatische Code Vervollständigung.

Automatisches vervollständigen von While, If, For, Func statements.


Die Suche wird mit Strg+F geöffnet.


Mit Strg+D wird der CodeFormatter gestartet, welcher den Code leserlicher gestaltet


Mit Strg+Klick auf eine Funktion oder Variable im Code Springt der Editor automatische zu dessen Definition (auch in anderen Dateien)


Mit Strg+Leer öffnet sich die Codevervollständigung. Mit Tab oder Enter, oder Doppelklick wird das gewählte Element an die Aktuelle stelle Eingefügt.


Wenn man nach einem While, If, For, Func eine Leerzeile einfügt (Enter Drücken) und das Statement wurde nicht abgeschlossen wird dies Automatisch hinzugefügt.


Die Funktionshilfe erscheint wenn man mindestens 2 Sekunden lang den Editorcursor auf einer Funktion hatte. Die Funtkionshilfe ist für die Standardfuntkionen bereits Definiert, für eigene Funktionen kann sie mit Doc Kommentaren über der Funktion hinzugefügt werden. Beispiel:

;*Doc Kommentare

;*Werden als Hilfe angezeigt

func myfunc()

 ...

endfunc
