Ereignisse

Formeditor ››

Der Ereignis Editor verknüpft Ereignisse von Controls mit Funktionen aus der Controller Datei.


Zu jedem Ereignisfeld gibt es eine Picklist, aus der der Nutzer aus allen Funktionen des Controllers eine für das Ereignis Aussuchen kann. Mit einem klick auf (Neu...) wird eine neue Funktion im Controller für dieses Ereignis erzeugt.


Ein Doppel (manchmal auch Dreifach) Klick auf das Editorfeld Springt automatisch zu der Funktion des Ereignisses. Ist noch keine Funktion angegeben, so wird eine neue Funktion dafür erstellt.
