Erster Start

Einführung ››

Bei dem ersten Start der IDE öffnet sich ein Fenster welches nach den Compiler Einstellungen Fragt:


In Debug Pfad und Release Pfad sind jeweils die Pfade zu den .exe Dateien des Compilers und Interpreters.


Log Pfad gibt an in welcher Datei die Kompilierungs/Interpreter Logs gesichert werden. ($ProjDir) wird dabei durch das Projektverzeichnis ersetzt.

IDE Ausgabe gibt an ob im Nachrichtenfenster der IDE die Ausgabe des Compilers oder des Interpreters angezeigt werden soll.

Komplette Ausgabe gibt auch noch den Syntaxtree im Nachrichtenfenster aus.


Wenn man alles ausgefüllt hat kann man mit Ok bestätigen und die IDE öffnet sich.
