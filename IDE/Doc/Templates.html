Templates

Projekte ››

Die Templates für Projekte befinden sich im Ordner Presets des IDE Verzeichnis.


Jedes Template wird durch ein einen Ordner mit Namen des Template im Presets Ordner repräsentiert.


In diesem Liegen zum einen alle Ordner und AAL Dateien die das Template benötigt, außerdem:

MainFile.apr: Diese Datei wird zur apr Datei des Projektes

FileList.txt: Diese Datei hält alle Dateien die das Projekt enthält, pro Datei eine Zeile

Icon.png: Ein 32x32 Pixel großes Icon welche das Template im Neues Projekt Fenster repräsentiert

PresetInfo.txt: Enthält alle wichtigen Informationen für das Preset im Format: Key=Wert z.B.

Als Keys gibt es:

MainFile: Name der apr Datei (meißt MainFile.apr)

OpendFile: Welche Dateien bei neuem Projekt direkt geöffnet sein sollen (mehrere Vorkommen erlaubt)

FocusedFile: Der Index der geöffneten Datei, deren Tab offen sein soll (0 = erstes Vorkommen von OpendFile)

AppType: Entweder GUI oder Console, ob es sich um eine GUI Anwendung Handelt

MainForm: Falls AppType GUI ist, muss hier die afm Datei des Hauptformulares angegeben werden.
