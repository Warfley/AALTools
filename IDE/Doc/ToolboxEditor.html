Toolbox & Editor

Formeditor ››

Der Formeditor ist ein Editor zum Bearbeiten von AAL Formularen (.afm). Zu jeder afm Datei wird eine Entsprechende .aal1 Datei (Controller) benötigt mit gleichem Namen.


Die Toolbox findet sich am unteren Rand des Formeditors und wird durch einen Klick auf Toolbox geöffnet oder geschlossen


Durch auswählen eines Tools wird bei Klick auf die Form diese Komponente erzeugt.


Die Komponenten können mit Drag&Drop Positioniert, vergrößert und verkleinert werden.

Alle Komponenten werden in einem Treeview auf der Rechten Seite angezeigt. Das im Treeview markierte Element ist repräsentiert die aktuell ausgewählte Komponente.


Durch drücken der Entf Taste wird das aktuell ausgewählte Element gelöscht.


Die Position kann per Drag&Drop auf dem kleinen Fenster unten Links im Formeditor bearbeitet werden.
